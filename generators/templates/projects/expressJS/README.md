### Visual Studio Code has *awesome* Markdown support!

* Split the editor (`Cmd+\`)
* Toggle preview (`Shift+CMD+V` on OSX, `Shift+Ctrl+V` on Windows and Linux)
* Press `Ctrl+Space` to see a list of Markdown snippets
